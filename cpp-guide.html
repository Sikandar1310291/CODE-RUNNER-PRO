1 Introduction to C++

What is C++?

C++ is a compiled, multi-paradigm, general-purpose programming language created by Bjarne Stroustrup in 1985. It extends C with Object-Oriented features, generic programming, and high-performance capabilities.

üéØ Why C++ Starts with 'C'?

It was originally called "C with Classes". The `++` operator means "increment by one" in C, symbolizing that C++ is the next evolution of C.

Why Use C++?

  • Performance: Used in Game Engines (Unreal), Browsers (Chrome), and OS kernels.
  • Control: Direct access to hardware and memory.
  • Scalability: Supports massive systems with millions of lines of code.

4 Pointers & Memory Management

This is what makes C++ powerful and dangerous. You manage memory manually.

Pointers vs References

Feature Pointer (*) Reference (&)
Definition Holds memory address Alias for an existing variable
Reassignable Yes No (Once set, it's fixed)
Null Value Can be nullptr Cannot be null
cpp 
int val = 10;
int* ptr = &val;  // Point to address of val
int& ref = val;   // Reference to val

*ptr = 20;        // Changes val to 20
ref = 30;         // Changes val to 30

5 Object-Oriented Programming

C++ fully supports OOP principles.

Classes and Objects

cpp 
class Car {
public: // Access specifier
    string brand;
    
    // Constructor
    Car(string b) {
        brand = b;
    }
    
    void honk() {
        cout << "Beep beep!" << endl;
    }
};

int main() {
    Car myCar("Tesla");
    myCar.honk();
    return 0;
}

Inheritance

cpp 
class Vehicle {
public:
    void move() { cout << "Moving..."; }
};

// Car inherits from Vehicle
class Bike : public Vehicle {
};

int main() {
    Bike b;
    b.move(); // Works!
}

6 Standard Template Library (STL)

Don't reinvent the wheel. Use STL for standard data structures.

Vectors (Dynamic Arrays)

cpp 
#include <vector>

vector<int> numbers = {1, 2, 3};
numbers.push_back(4); // Add to end inside [1, 2, 3, 4]
numbers.pop_back();   // Remove last

Maps (Key-Value)

cpp 
#include <map>

map<string, int> ages;
ages["Alice"] = 25;
ages["Bob"] = 30;

cout << ages["Alice"]; // 25

7 Modern C++ Features (C++11/14/17/20)

Smart Pointers

Avoid memory leaks by using Smart Pointers instead of `new`/`delete`.

cpp 
#include <memory>

// unique_ptr: Automatically deleted when out of scope
unique_ptr<int> ptr = make_unique<int>(10);

// shared_ptr: Shared ownership
shared_ptr<int> sptr = make_shared<int>(20);

Lambda Expressions

cpp 
auto add = [](int a, int b) {
    return a + b;
};

cout << add(5, 3); // 8
