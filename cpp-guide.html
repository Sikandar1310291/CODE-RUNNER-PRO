2 Classes & Objects

Classes are blueprints for objects. They group data and methods together.

cpp 
#include <iostream>
#include <string>

class Car {
public:
    std::string brand;
    void drive() {
        std::cout << brand << " is driving!" << std::endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Tesla";
    myCar.drive();
    return 0;
}

6 Mastering the STL

The Standard Template Library (STL) provides generic classes for common tasks.

cpp 
#include <vector>
#include <algorithm>

std::vector<int> numbers = {5, 2, 8, 1};

// Sorting using STL algorithms
std::sort(numbers.begin(), numbers.end());

// Iterating
for(int n : numbers) {
    std::cout << n << " ";
}

7 Memory Management: Smart Pointers

Modern C++ discourages raw new and delete. Use Smart Pointers to avoid memory leaks.

cpp 
#include <memory>

// Automatically deallocated when out of scope
std::unique_ptr<int> val = std::make_unique<int>(42);

// Shared ownership between multiple pointers
std::shared_ptr<int> ptr1 = std::make_shared<int>(100);
std::shared_ptr<int> ptr2 = ptr1;
