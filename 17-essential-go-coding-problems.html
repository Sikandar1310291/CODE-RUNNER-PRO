4. Prime Check

Problem: Implementation of Prime Check algorithm in Go.

Run Live 
func isPrime(n int) bool {
  if n < 2 { return false }
  for i := 2; i*i <= n; i++ {
    if n%i == 0 { return false }
  }
  return true
}

Optimized prime test in Go.

Time Complexity

O(sqrt(n))

Space Complexity

O(1)
