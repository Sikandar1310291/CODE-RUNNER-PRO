6. Binary Search

Problem: Implementation of Binary Search algorithm in PHP.

Run Live 
function bs($arr, $x) {
  $l = 0; $r = count($arr)-1;
  while($l <= $r) {
    $m = floor(($l+$r)/2);
    if($arr[$m] == $x) return $m;
    if($arr[$m] < $x) $l = $m+1; else $r = $m-1;
  }
  return -1;
}

Iterative search implementation.

Time Complexity

O(log n)

Space Complexity

O(1)
