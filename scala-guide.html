1 Introduction to Scala

Scala stands for "Scalable Language". It runs on the Java Virtual Machine (JVM) and is fully interoperable with Java. If you know Java, you can use all your favorite libraries in Scala.

scala 
object HelloWorld extends App {
    println("Hello, World!")
}

2 Variables & Functions

Scala prefers immutability. Use val for constants and `var` for variables.

scala 
val pi = 3.14  // Cannot be changed (Immutable)
var age = 25   // Can be changed (Mutable)

// Function definition
def add(a: Int, b: Int): Int = a + b

3 Functional Collections

Map, Filter, and Reduce are core to Scala programming.

scala 
val nums = List(1, 2, 3, 4, 5)

val doubled = nums.map(_ * 2)  // List(2, 4, 6, 8, 10)
val even = nums.filter(_ % 2 == 0) // List(2, 4)

4 Case Classes & Matching

Case classes are like Java records but more powerful.

scala 
case class Person(name: String, age: Int)

val bob = Person("Bob", 30)

// Pattern Matching
bob match {
    case Person("Bob", _) => println("Hi Bob!")
    case Person(name, age) => println(s"Hi $name")
}

5 Asynchronous Futures

Handle long-running tasks without blocking the main thread.

scala 
import scala.concurrent.Future
import scala.concurrent.ExecutionContext.Implicits.global

val f = Future {
    Thread.sleep(1000)
    "Result ready!"
}
