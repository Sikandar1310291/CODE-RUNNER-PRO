4 Control Flow in R

Conditional Statements

r 
year <- 2024

if (year >= 2020) {
  print("Modern R")
} else {
  print("Older R")
}

Note: R uses curly braces {} instead of indentation to define Vectorized Conditional Logic with ifelse()

r 
x <- 8
result <- ifelse(x > 5, "Large", "Small")
print(result)  # Output: "Large"

This is commonly used in data analysis workflows.

for Loop

r 
for (i in 1:5) {
  print(i)
}

while Loop

r 
count <- 1
while (count <= 5) {
  print(count)
  count <- count + 1
}

5 Data Structures in R

Data structures are one of R's greatest strengths.

Vectors

r 
numbers <- c(1, 2, 3, 4)
names <- c("Ali", "Sara", "John")

R is a vector-based language, meaning most operations are applied to entire vectors at once.

Matrices

r 
m <- matrix(1:6, nrow = 2)
print(m)

Matrices store data of the same type in two dimensions.

Lists

r 
info <- list(name = "Ali", age = 20, active = TRUE)
print(info$name)  # Access by name

Lists can store different data types.

Data Frames

r 
df <- data.frame(
  name = c("Ali", "Sara"),
  age = c(20, 22),
  city = c("NYC", "LA")
)
print(df)

Data frames represent real-world tabular datasets.

7 Basic Statistical Concepts in R

Statistics is at the core of R.

Descriptive Statistics

r

These functions help understand data distribution and trends.

Factors (Categorical Data)

r

Factors are essential for categorical analysis and modeling.

Apply Family Functions

r

sapply(numbers, sqrt)  # Returns vector

These functions provide efficient alternatives to loops.

8 Error Handling in R

Common Types of Errors

  • Syntax errors
  • Runtime errors
  • Logical errors

Handling Errors with tryCatch()

r 
tryCatch(
  {
    log("abc")  # This will cause an error
  },
  error = function(e) {
    print(paste("Error:", e$message))
  },
  finally = {
    print("Execution complete")
  }
)

tryCatch is essential for robust scripts that run automatically.

9 File Handling in R

File handling is the foundation of real-world data analysis in R.

Reading Text Files

r 
data <- readLines("file.txt")

Used for simple unstructured text data.

Reading CSV Files

r 
df <- read.csv("data.csv")
head(df)  # View first 6 rows

CSV is the most commonly used data format in analytics.

Writing CSV Files

r 
write.csv(df, "output.csv", row.names = FALSE)

Setting row.names = FALSE is a professional best practice.

Saving R Objects (RDS Format)

r 
saveRDS(df, "data.rds")

RDS files advantages:

  • ‚úÖ Preserve object structure
  • ‚úÖ Load faster than CSV
  • ‚úÖ Reduce parsing overhead

Loading R Objects

Managing the Working Directory

A correct working directory prevents file-not-found errors.
