17 Essential Python Coding Problems

Boost your logic and interview skills with these foundational Python challenges.

Python is known for its readability and simplicity. These 17 problems cover fundamental algorithms and data structures that every Python developer should master.

Whether you're preparing for a technical interview or just starting your journey in Python, mastering these 17 problems will give you a solid foundation in algorithm design and problem-solving. Each solution includes an interactive "Run Live" button so you can experiment with the code yourself!

2. Factorial (recursive)

Problem: Implementation of Factorial (recursive) algorithm in Python.

Run Live 
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

print(factorial(5))

A function that calls itself until it reaches the base case (n=0).

Time Complexity

O(n)

Space Complexity

O(n) due to recursion stack

3. Fibonacci

Problem: Implementation of Fibonacci algorithm in Python.

Run Live 
def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)

print([fib(i) for i in range(10)])

Computes the nth Fibonacci number. Every number is the sum of the two preceding ones.

Time Complexity

O(2^n)

Space Complexity

O(n)

4. Prime Check

Problem: Implementation of Prime Check algorithm in Python.

Run Live 
def is_prime(n):
    if n < 2: return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

print(is_prime(17))

Checks if a number is divisible only by 1 and itself by testing divisors up to the square root.

Time Complexity

O(sqrt(n))

Space Complexity

O(1)

6. Binary Search

Problem: Implementation of Binary Search algorithm in Python.

Run Live 
def binary_search(arr, x):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] < x:
            low = mid + 1
        elif arr[mid] > x:
            high = mid - 1
        else:
            return mid
    return -1

print(binary_search([1, 2, 3, 4, 5], 4))

Efficiently finds an element in a sorted array by repeatedly dividing the search interval in half.

Time Complexity

O(log n)

Space Complexity

O(1)

8. Power Function

Problem: Implementation of Power Function algorithm in Python.

Run Live 
def power(base, exp):
    if exp == 0: return 1
    res = power(base, exp // 2)
    if exp % 2 == 0:
        return res * res
    return res * res * base

print(power(2, 10))

Calculates base^exp using binary exponentiation, significantly faster than linear multiplication.

Time Complexity

O(log n)

Space Complexity

O(log n)

13. Factorial (iterative)

Problem: Implementation of Factorial (iterative) algorithm in Python.

Run Live 
def factorial_iter(n):
    res = 1
    for i in range(2, n + 1):
        res *= i
    return res

print(factorial_iter(5))

Calculates factorial using a simple loop, avoiding recursion limits and overhead.

Time Complexity

O(n)

Space Complexity

O(1)

14. Armstrong Number

Problem: Implementation of Armstrong Number algorithm in Python.

Run Live 
def is_armstrong(n):
    s = str(n)
    p = len(s)
    return n == sum(int(digit)**p for digit in s)

print(is_armstrong(153))

Checks if the sum of digits raised to the power of number of digits equals the number itself.

Time Complexity

O(d) where d is digits

Space Complexity

O(d)
