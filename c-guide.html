1 Introduction to C

What is C?

C is a general-purpose, procedural programming language supporting structured programming, lexical variable scope, and recursion. It was developed by Dennis Ritchie in 1972 at Bell Labs.

üéØ Why is C Still Used?

  • OS Development: Windows, Linux, and macOS kernels are written in C.
  • Embedded Systems: Microcontrollers like Arduino use C/C++.
  • Speed: C allows low-level access to memory with minimal overhead.

7 Pointers

Pointers store the address of another variable. They are the strongest feature of C.

c 
int val = 20;
int *ptr = &val;  // ptr holds the address of val

printf("Value: %d\n", val);
printf("Address: %p\n", ptr);
printf("Dereferenced: %d\n", *ptr); // Access value via pointer

8 Arrays & Strings

Strings in C are just arrays of characters terminated by a special `\0` character.

c 
int numbers[] = {1, 2, 3, 4};
char name[] = "Hello"; 

printf("%s", name); // Prints "Hello"

9 Structures

Structs allow you to group related variables.

c 
struct Point {
    int x;
    int y;
};

int main() {
    struct Point p1;
    p1.x = 10;
    p1.y = 20;
    
    printf("Point: (%d, %d)", p1.x, p1.y);
    return 0;
}
