1 Introduction to Go

Go was created at Google to solve problems of scale. It combines the efficiency of C++ with the readability of Python. It's the language behind Docker, Kubernetes, and Terraform.

go 
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

2 Variables & Types

Go is statically typed, but has type inference using :=.

go 
var name string = "Alice" // Explicit
age := 30                 // Inferred (int)

const Pi = 3.14           // Constant

3 Control Structures

Go has only one looping construct: for.

go 
// Standard Loop
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// While-like Loop
i := 0
for i < 5 {
    i++
}

// If-Else
if x > 10 {
    fmt.Println("Big")
} else {
    fmt.Println("Small")
}

4 Functions

Go functions can return multiple values, which is often used for error handling.

go 
func swap(x, y string) (string, string) {
    return y, x
}

func main() {
    a, b := swap("hello", "world")
    fmt.Println(a, b) // world hello
}

5 Structs & Interfaces

Go doesn't have classes. It uses Structs for data and Interfaces for behavior.

go 
type User struct {
    Name  string
    Email string
}

// Method on User
func (u User) Greet() {
    fmt.Println("Hi, I am", u.Name)
}

6 Concurrency

This is Go's superpower. Use go to start a lightweight thread.

Goroutines

Goroutines are functions that run concurrently. They are cheaper than OS threads.

go 
func say(s string) {
    for i := 0; i < 3; i++ {
        fmt.Println(s)
    }
}

func main() {
    go say("world") // Runs in background
    say("hello")    // Runs in foreground
}

7 Error Handling

Go doesn't use Try/Catch. Errors are just values returned by functions.

go 
file, err := os.Open("file.txt")
if err != nil {
    log.Fatal(err)
}
// Do something with file
