1 Introduction to Swift

What is Swift?

Swift is a compiled programming language developed by Apple for iOS, macOS, watchOS, tvOS, and Linux. It effectively replaces Objective-C with a modern, safe, and readable syntax.

üéØ Why Swift?

  • Modern: Released in 2014, it lacks the baggage of older C-based languages.
  • Safe: Default variables cannot be null (nil), preventing crashes.
  • Open Source: Supported by a huge community beyond just Apple.

2 Basic Syntax & Types

Swift uses let for constants and var for variables.

swift 
print("Hello, World!") // No semicolon needed!

let pi = 3.14159 // Constant (cannot change)
var age = 25     // Variable (can change)

// Type annotations (optional)
var name: String = "Alice"
var isValid: Bool = true

3 Optionals

The most unique feature of Swift. An Optional allows a variable to hold a value OR be nil.

swift 
var name: String? = "Alice" // Can contain String or nil
name = nil // Valid

// Safe Unwrapping (if-let)
if let safeName = name {
    print("Hello, \(safeName)")
} else {
    print("Name is missing")
}

4 Control Flow

For-In Loop

swift 
let names = ["Anna", "Alex", "Brian"]

}

// Ranges

    print(i) // 1, 2, 3, 4, 5
}

6 Structs vs Classes

Swift prefers Structs (Value Types) over Classes (Reference Types) for most data models.

Feature Struct Class
Type Value Type (Copied) Reference Type (Shared)
Inheritance No Yes
Usage Models, Data Managers, UIView
swift 
struct User {
    var name: String
    var age: Int
}

var u1 = User(name: "A", age: 10)
var u2 = u1 // u2 is a COPY of u1
u2.age = 20
print(u1.age) // 10 (u1 did not change)

7 Protocols

Similar to Interfaces in other languages, but more powerful.

swift 
protocol Drivable {
    var speed: Int { get }
    func drive()
}

struct Car: Drivable {
    var speed: Int = 100
    func drive() {
        print("Vroom!")
    }
}
