4. Prime Check

Problem: Implementation of Prime Check algorithm in Rust.

Run Live 
fn is_prime(n: u64) -> bool {
  if n < 2 { return false }
  for i in 2..=((n as f64).sqrt() as u64) {
    if n % i == 0 { return false }
  }
  true
}

Range-based prime test.

Time Complexity

O(sqrt(n))

Space Complexity

O(1)
