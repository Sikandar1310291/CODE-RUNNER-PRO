1 Introduction to Java

What is Java?

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. The core principle of Java is "Write Once, Run Anywhere" (WORA), meaning Java code compiled on one platform (like Windows) can run on any other (like Linux or MacOS) without recompilation.

History of Java

Java was started in 1991 by James Gosling and his team at Sun Microsystems (now acquired by Oracle). It was originally called Oak, but later renamed to Java (after the coffee). The first version, Java 1.0, was released in 1996.

üéØ Why is Java So Popular?

  • Platform Independent: Runs on the Java Virtual Machine (JVM).
  • Object-Oriented: Encourages clean, modular, and reusable code.
  • Secure: Strong memory management and lack of explicit pointers.
  • Multithreaded: Built-in support for concurrent execution.
  • Robust Enterprise Support: The backbone of many banking, insurance, and retail systems.

2 Environment Setup

To write and run Java code, you need the Java Development Kit (JDK).

Installation Steps

  1. Download the latest JDK (JDK 17 or 21 recommended) from the Oracle website or OpenJDK.
  2. Install the JDK and set the JAVA_HOME environment variable.
  3. Verify the installation in your terminal:
bash 
java -version
javac -version

Recommended IDEs

While you can use Notepad, professional developers use Intelligent IDEs:

  • IntelliJ IDEA (Most popular, highly recommended)
  • Eclipse (Classic, widely used in legacy systems)
  • VS Code (Lightweight, great with Extension Pack for Java)

3 Basic Syntax & Structure

Let's look at a simple "Hello, World!" program. In Java, everything must reside inside a class.

java 
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Code Breakdown

  • public class Main: Defines a class named Main. The file name must be Main.java.
  • public static void main(String[] args): The main method. This is the entry point where the JVM starts execution.
  • System.out.println(): Prints text to the console.

5 Control Flow

If-Else & Switch

java 
int time = 20;

if (time < 12) {
    System.out.println("Good morning");
} else if (time < 18) {
    System.out.println("Good afternoon");
} else {
    System.out.println("Good evening");
}

Modern Switch (Java 14+)

java 
String day = "Monday";
switch (day) {
    case "Monday", "Friday" -> System.out.println("Work day");
    case "Saturday", "Sunday" -> System.out.println("Weekend");
    default -> System.out.println("Midweek");
}

6 Object-Oriented Programming (OOP)

OOP is the heart of Java. It organizes software design around objects (data) rather than function and logic.

1. Classes & Objects

A class is a blueprint; an object is an instance of that blueprint.

java 
class Car {
    String brand;
    
    // Constructor
    Car(String brand) {
        this.brand = brand;
    }
    
    void honk() {
        System.out.println("Beep beep!");
    }
}

// Usage
Car myCar = new Car("Tesla");
myCar.honk();

2. Inheritance

Allows a class to acquire properties and methods of another class using extends.

java 
class Vehicle {
    void go() { System.out.println("Moving..."); }
}

class Bike extends Vehicle {
    // Inherits go() from Vehicle
}

Bike b = new Bike();
b.go(); // Output: Moving...

3. Encapsulation

Hiding data using private and accessing it via getters/setters.

4. Polymorphism

Objects taking on multiple forms (e.g., Method Overriding).

7 Collections Framework

Java provides powerful data structures in the java.util package.

ArrayList (Dynamic Array)

java 
import java.util.ArrayList;

ArrayList list = new ArrayList<>();
list.add("Java");
list.add("Python");
list.remove("Python");
System.out.println(list.get(0)); // Output: Java

HashMap (Key-Value Pairs)

java 
import java.util.HashMap;

HashMap scores = new HashMap<>();
scores.put("John", 90);
scores.put("Alice", 85);

System.out.println(scores.get("John")); // Output: 90

8 Exception Handling

Java uses try-catch-finally to handle runtime errors without crashing the program.

java 
try {
    int[] nums = {1, 2, 3};
    System.out.println(nums[10]); // Index out of bounds
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Something went wrong!");
} finally {
    System.out.println("This always runs.");
}

9 File Handling

Reading and writing files is essential. Here is the modern way using FileWriter.

java 
import java.io.FileWriter;
import java.io.IOException;

try {
    FileWriter writer = new FileWriter("output.txt");
    writer.write("Hello from Java!");
    writer.close();
    System.out.println("File written successfully.");
} catch (IOException e) {
    e.printStackTrace();
}
