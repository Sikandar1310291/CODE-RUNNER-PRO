1 Introduction to JavaScript

What is JavaScript?

JavaScript (JS) is a high-level, interpreted programming language that conforms to the ECMAScript specification. It is the language of the web, enabling interactive web pages and is an essential part of web applications alongside HTML and CSS.

üéØ Why JavaScript?

  • Versatile: Runs on client (browser) and server (Node.js).
  • Interactive: Creates dynamic content, animations, and forms.
  • Popular: Massive ecosystem with libraries like React, Vue, and Angular.

Linking JavaScript

You can add JavaScript directly in HTML or link an external file.

html 
<!-- External File (Recommended) -->
<script src="script.js"></script>

<!-- Internal Script -->
<script>
    console.log("Hello, World!");
</script>

2 Variables & Data Types

Declaring Variables

Modern JavaScript uses let and const instead of the legacy var.

Keyword Scope Reassignable? Use Case
const Block No Values that won't change (e.g., config, API keys)
let Block Yes Variables that will change (e.g., counters, user input)
var Function Yes Legacy code (Avoid using in modern projects)
javascript 
let name = "Alice";      // String
const age = 25;          // Number
let isStudent = true;    // Boolean
let emptyValue = null;   // Null (intentional empty)
let notDefined;          // Undefined (not initialized)

3 Functions & Arrow Functions

Traditional Function

javascript 
function greet(name) {
    return "Hello, " + name;
}

Arrow Functions (ES6)

Arrow functions provide a shorter syntax and safer this binding.

javascript 
const greet = (name) => {
    return `Hello, ${name}`;
};

// Shorthand for single line return
const add = (a, b) => a + b;

6 Asynchronous JavaScript

Handling operations that take time (like API calls) without freezing the browser.

Fetch API with Async/Await

javascript 
async function getUserData() {
    try {
        const response = await fetch("https://api.example.com/user");
        if (!response.ok) throw new Error("Network response was not ok");
        
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error("Error fetching data:", error);
    }
}

7 Modern ES6+ Features

Destructuring

javascript 
const user = { id: 1, name: "Alice" };
const { name } = user; // Extracts name

const colors = ["Red", "Blue"];
const [first, second] = colors;

Spread & Rest Operators

javascript 
// Spread: Copies properties/elements
const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4]; // [1, 2, 3, 4]

// Rest: Gathers arguments
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

8 LocalStorage

Store data in the user's browser that persists even after refreshing.

javascript 
// Saving data
localStorage.setItem("theme", "dark");

// Retrieving data
const theme = localStorage.getItem("theme");

// Saving Objects (must be stringified)
const settings = { volume: 50, notifications: true };
localStorage.setItem("settings", JSON.stringify(settings));

// Retrieving Objects
const savedSettings = JSON.parse(localStorage.getItem("settings"));

9 Error Handling

Prevent your app from crashing using try...catch.

javascript 
try {
    // Code that might fail
    let result = riskyOperation();
    console.log(result);
} catch (error) {
    // Handling the error
    console.error("An error occurred:", error.message);
} finally {
    // Runs regardless of success or failure
    console.log("Operation complete.");
}
