4. Prime Check

Problem: Implementation of Prime Check algorithm in C.

Run Live 
int isPrime(int n) {
  if(n<2) return 0;
  for(int i=2; i*i<=n; i++) if(n%i==0) return 0;
  return 1;
}

Divide and conquer primality check.

Time Complexity

O(sqrt(n))

Space Complexity

O(1)
