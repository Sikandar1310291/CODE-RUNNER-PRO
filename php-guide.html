4 Secure Database (PDO)

Never use mysql_query. Always use PDO with prepared statements to prevent SQL injection.

php 
$stmt = $pdo->prepare('SELECT * FROM users WHERE email = ?');
$stmt->execute([$email]);
$user = $stmt->fetch();
