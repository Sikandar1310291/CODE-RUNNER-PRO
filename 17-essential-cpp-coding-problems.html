4. Prime Check

Problem: Implementation of Prime Check algorithm in C++.

Run Live 
bool isPrime(int n) {
  if(n < 2) return false;
  for(int i=2; i*i<=n; i++) if(n%i==0) return false;
  return true;
}

Looping up to sqrt(n) for fast prime detection.

Time Complexity

O(sqrt(n))

Space Complexity

O(1)

14. Armstrong Number

Problem: Implementation of Armstrong Number algorithm in C++.

Run Live 
bool isArm(int n) {
  int s=0, t=n, d=std::to_string(n).length();
  while(t) { s += std::pow(t%10, d); t/=10; }
  return s==n;
}

Loop-based power aggregation of digits.

Time Complexity

O(d)

Space Complexity

O(1)
