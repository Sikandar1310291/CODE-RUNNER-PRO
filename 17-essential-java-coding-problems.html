17 Essential Java Coding Problems

Boost your logic and interview skills with these foundational Java challenges.

Java's strong typing and OOP principles are essential for scalable apps. Here are 17 problems implemented in modern Java style.

Whether you're preparing for a technical interview or just starting your journey in Java, mastering these 17 problems will give you a solid foundation in algorithm design and problem-solving. Each solution includes an interactive "Run Live" button so you can experiment with the code yourself!

2. Factorial (recursive)

Problem: Implementation of Factorial (recursive) algorithm in Java.

Run Live 
public class Main {
  static int fact(int n) { return n<=1 ? 1 : n * fact(n-1); }
  public static void main(String[] args) { System.out.println(fact(5)); }
}

Static recursive method for factorial calculation.

Time Complexity

O(n)

Space Complexity

O(n)

3. Fibonacci

Problem: Implementation of Fibonacci algorithm in Java.

Run Live 
public class Main {
  static int fib(int n) { return n<2 ? n : fib(n-1)+fib(n-2); }
  public static void main(String[] args) { System.out.println(fib(7)); }
}

Classic recursive Fibonacci implementation.

Time Complexity

O(2^n)

Space Complexity

O(n)

4. Prime Check

Problem: Implementation of Prime Check algorithm in Java.

Run Live 
public class Main {
  public static void main(String[] args) {
    int n = 17; boolean p = true;
    for(int i=2; i<=Math.sqrt(n); i++) if(n%i==0) p=false;
    System.out.println(p);
  }
}

Looping through divisors up to the square root for efficiency.

Time Complexity

O(sqrt(n))

Space Complexity

O(1)

7. GCD

Problem: Implementation of GCD algorithm in Java.

Run Live 
public class Main {
  static int gcd(int a, int b) { return b==0 ? a : gcd(b, a%b); }
  public static void main(String[] args) { System.out.println(gcd(48, 18)); }
}

Euclidean GCD algorithm in Java.

Time Complexity

O(log(min(a,b)))

Space Complexity

14. Armstrong Number

Problem: Implementation of Armstrong Number algorithm in Java.

Run Live 
public class Main {
  public static void main(String[] args) {
    int n=153, temp=n, sum=0, p=String.valueOf(n).length();
    while(temp>0) { sum+=Math.pow(temp%10, p); temp/=10; }
    System.out.println(sum==n);
  }
}

Calculates digit powers using a while loop.

Time Complexity

O(d)

Space Complexity

O(1)

17. Even/Odd Count

Problem: Implementation of Even/Odd Count algorithm in Java.

Run Live 
public class Main {
  public static void main(String[] args) {
    int[] arr = {1, 2, 3, 4, 5};
    long e = Arrays.stream(arr).filter(x -> x%2==0).count();
    System.out.println("Even: "+e+", Odd: "+(arr.length-e));
  }
}

Stream counting for even and odd distribution.

Time Complexity

O(n)

Space Complexity

O(1)
