Sample Java programs you can try

Try these examples to get comfortable with the Java syntax.

1. Hello World

The standard entry point for any Java program.

public class Main { public static void main(String[] args) { System.out.println("Hello, World!"); } }

2. Basic Arithmetic

Performing calculations with variables.

public class Main { public static void main(String[] args) { int a = 10; int b = 20; int sum = a + b; System.out.println("The sum is: " + sum); } }

3. Using Loops

A simple for loop execution.

public class Main { public static void main(String[] args) { for (int i = 1; i <= 5; i++) { System.out.println("Counter: " + i); } } }

4. Java Functions (Methods)

Defining and calling a static method.

public class Main { static void greet(String name) { System.out.println("Hello " + name); } public static void main(String[] args) { greet("Alice"); greet("Bob"); } }

FAQ: Java online compiler

Do I need to install Java?

No, our cloud environment handles the Java Runtime Environment (JRE) for you.

What is the class name requirement?

For the main entry point, please define your class as public class Main so our runner can identify the entry point.

Can I use Java Collections?

Yes, you can import java.util.* and use lists, maps, sets, and other collection frameworks.

Is it free?

Yes, CodeRunner Pro's Java compiler is 100% free for everyone.

Can I run Swing or JavaFX code?

No, online compilers typically support console (text) output only. GUI frameworks like Swing require a local desktop environment.
