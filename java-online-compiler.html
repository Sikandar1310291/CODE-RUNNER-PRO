Frequently Asked Questions

Maven/Gradle support?
Current support is for single-file standard Java libraries.
Java 8 vs 21?
We use JDK 21 for modern features like switch patterns and virtual threads.
Memory limits?
Default JVM heap is limited to 512MB for stability.
Can I run GUI apps?
Headless execution only; GUI (Swing/AWT) is not supported in the cloud.
