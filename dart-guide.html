2 Sound Null Safety

Dart 3 is 100% null safe. This means variables cannot be null unless you explicitly allow it using the ? operator.

dart 
String name = "Sikandar"; // Cannot be null
String? nickName;        // Can be null

void main() {
  print(name.length);    // Safe
  // print(nickName.length); // COMPILE ERROR
  print(nickName?.length ?? 0); // Safe unwrap
}

5 Asynchronous Programming

Dart handles concurrency using an event loop. Future represents a single value, while Stream represents a series of values.

dart 
Future fetchUser() async {
  await Future.delayed(Duration(seconds: 2));
  return "Flutter Dev";
}

void main() async {
  print("Fetching...");
  String user = await fetchUser();
  print("Hello $user");
}
