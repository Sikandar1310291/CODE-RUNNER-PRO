1 Introduction to Dart

Dart is a client-optimized language for fast apps on any platform.

dart 
void main() {
  print('Hello, World!');
}

2 Variables & Types

Dart is type-safe. You can use var or explicit types.

dart 
var name = 'Voyager I';
var year = 1977;
var antennaDiameter = 3.7;
var flybyObjects = ['Jupiter', 'Saturn', 'Uranus', 'Neptune'];
var image = {
  'tags': ['saturn'],
  'url': '//path/to/saturn.jpg'
};

3 Functions

Functions are objects and have a type, Function.

dart 
int fibonacci(int n) {
  if (n == 0 || n == 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

var result = fibonacci(20);

4 Classes & OOP

Dart is an object-oriented language with classes and mixin-based inheritance.

dart 
class Spacecraft {
  String name;
  DateTime? launchDate;

  // Constructor
  Spacecraft(this.name, this.launchDate) {
    // Initialization code goes here.
  }

  void describe() {
    print('Spacecraft: $name');
  }
}

5 Null Safety

In Dart, variables cannot be null by default. You must declare them as nullable using ?.

dart 
String? name; // Nullable type. Can be `null` or string.
String name2; // Non-nullable type. Cannot be `null`.

name = null; // OK
// name2 = null; // Error!
