1 Introduction to Rust

Rust is a systems programming language that empowers everyone to build reliable and efficient software. It guarantees memory safety and thread safety at compile-time.

rust 
fn main() {
    println!("Hello, world!");
}

2 Ownership & Borrowing

This is the most important concept in Rust. Every value has an owner.

üìù Rules of Ownership

  • Each value in Rust has a variable that‚Äôs called its owner.
  • There can only be one owner at a time.
  • When the owner goes out of scope, the value will be dropped.
rust 
let s1 = String::from("hello");
let s2 = s1; 
// println!("{}", s1); // Error! s1 no longer valid (ownership moved)

// Borrowing using references
let len = calculate_length(&s2); // passed reference

3 Variables & Mutability

5 Pattern Matching

Rust has excellent support for pattern matching with match.

rust 
let number = 3;

match number {
    1 => println!("One"),
    2 => println!("Two"),
    3 => println!("Three"),
    _ => println!("Other"),
}
