1 Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is not a programming language, but a markup language used to create the structure of web pages.

üéØ The Trio of Web Dev

  • HTML: Structure (The skeleton)
  • CSS: Style (The clothes/makeup)
  • JavaScript: Functionality (The brain/muscle)

2 Document Structure

Every HTML5 document follows a specific boilerplate structure.

html 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
</head>
<body>
    <h1>Welcome to my site!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

7 Semantic HTML

Using tags that describe their meaning improves accessibility and SEO.

Why Semantic?

Instead of using <div> for everything, use these specific tags:

  • <header>: Top section of page/article.
  • <nav>: Navigation links.
  • <main>: Main content area.
  • <footer>: Footer content.
  • <article>: Self-contained content like a blog post.
