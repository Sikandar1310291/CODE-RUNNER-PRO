4. Prime Check

Problem: Implementation of Prime Check algorithm in Dart.

Run Live 
bool isPrime(int n) {
  if(n < 2) return false;
  for(var i=2; i*i<=n; i++) if(n%i==0) return false;
  return true;
}

Looping divisors to square root.

Time Complexity

O(sqrt(n))

Space Complexity

O(1)
