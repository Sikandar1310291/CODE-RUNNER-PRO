2. Factorial (recursive)

Problem: Implementation of Factorial (recursive) algorithm in SQL.

Run Live 
WITH RECURSIVE fact(n, v) AS (
  SELECT 1, 1 UNION ALL
  SELECT n + 1, (n + 1) * v FROM fact WHERE n < 5
) SELECT v FROM fact WHERE n = 5;

CTE recursive factorial generation.

Time Complexity

O(n)

Space Complexity

O(n)

3. Fibonacci

Problem: Implementation of Fibonacci algorithm in SQL.

Run Live 
WITH RECURSIVE fib(n, a, b) AS (
  SELECT 1, 0, 1 UNION ALL
  SELECT n + 1, b, a + b FROM fib WHERE n < 10
) SELECT a FROM fib;

Recursive sequence generation.

Time Complexity

O(n)

Space Complexity

O(n)

4. Prime Check

Problem: Implementation of Prime Check algorithm in SQL.

Run Live
