1 Introduction to TypeScript

TypeScript is a superset of JavaScript that adds static types. Browsers just read it as normal JavaScript after compilation.

typescript 
// TypeScript
const hello: string = "Hello, World!";

// Compiles to JavaScript:
// const hello = "Hello, World!";

2 Basic Types

Define exactly what kind of data a variable holds.

typescript 
let isDone: boolean = false;
let lines: number = 42;
let name: string = "Anders";

// Arrays
let list: number[] = [1, 2, 3];

// Any (Opt-out of type checking)
let notSure: any = 4;

3 Interfaces

Interfaces define the shape of an object.

typescript 
interface User {
    id: number;
    name: string;
    email?: string; // Optional property
}

function registerUser(user: User) {
    console.log("Registered: " + user.name);
}

4 Generics

Create components that work with a variety of types rather than a single one.

typescript 
function identity(arg: T): T {
    return arg;
}

let output = identity("myString");
