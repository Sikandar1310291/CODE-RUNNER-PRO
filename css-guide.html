1 Introduction: The Power of Style

CSS (Cascading Style Sheets) is the language used to describe the presentation of a document written in HTML. It controls everything from the colors and fonts to the complex layouts and animations that make a website feel "alive."

4 Flexbox

Flexbox is a layout module that makes it easy to align items in a row or column, even when their size is unknown.

css 
.container {
  display: flex;
  justify-content: space-between; /* Space out items */
  align-items: center;            /* Center vertically */
  gap: 20px;                      /* Easy spacing */
}

6 CSS Variables

Custom properties allow you to define values once and reuse them throughout your stylesheet. Perfect for Dark Mode!

css 
:root {
  --primary-color: #264de4;
  --text-main: #333;
}

.button {
  background: var(--primary-color);
  color: white;
}
