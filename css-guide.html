1 Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets. If HTML is the skeleton, CSS is the skin, makeup, and clothing. It controls the color, layout, and fonts of your website.

css 
/* Selecting all Paragraphs */
p {
    color: red;
    font-size: 16px;
}

/* Selecting by Class */
.btn-primary {
    background-color: blue;
    color: white;
}

3 The Box Model

Every element in CSS is a box. Understanding this is key to layout.

üì¶ The Layers

  • Content: The text or image itself.
  • Padding: Space INSIDE the border.
  • Border: A line around the padding.
  • Margin: Space OUTSIDE the border (pushes neighbors away).

4 Flexbox

Flexbox is a layout module designed for locating items in rows or columns.

css 
.container {
    display: flex;
    justify-content: center; /* Horizonal Alignment */
    align-items: center;     /* Vertical Alignment */
}

5 CSS Grid

Grid is for two-dimensional layouts (rows AND columns at once).

css 
.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* 3 Equal Columns */
    gap: 20px;
}

6 Responsive Design

Using Media Queries to change styles based on screen width.

css 
.sidebar {
    display: block;
}

/* On Mobile Screens (small) */
@media (max-width: 768px) {
    .sidebar {
        display: none; /* Hide sidebar on mobile */
    }
}
